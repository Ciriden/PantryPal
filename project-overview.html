About PantryPal

PantryPal is a user-friendly web application designed to help individuals manage recipes and ingredients efficiently. The platform allows users to create, manage, and delete recipes, organize ingredients, and track pantry inventory dynamically. Additionally, the application includes a shopping cart feature for adding missing ingredients and supports user accounts for personalized recipe management.

This project aims to develop an intuitive cooking companion that simplifies meal planning and grocery management for everyday home users.

Project Priorities

1. Recipe Management

  • Create, manage, and delete recipes with step-by-step instructions
  • Include amount values, measurement units, and ingredient names for each step
  • Options for tracking allergens and suggesting substitutions
  • Servings with scalable ingredient amounts
  • Prep time tracking with built-in cooking timer

2. Pantry Inventory

  • Dynamic inventory of all foods/ingredients
  • Automatic updates when using ingredients in recipes
  • Categorized organization by food types

3. Shopping Cart

  • Auto-generate ingredient lists from recipes
  • Manual addition option for other grocery items
  • Seamless integration with pantry inventory

4. Minimalist Design

  • Visual-first approach with emphasis on images
  • Simple, intuitive interface for everyday users
  • Clean layout with minimal text

5. User Accounts

  • Save and organize personal recipes
  • Track pantry inventory across sessions
  • Personalized cooking experience

6. Stretch Goals

  • Social features for sharing recipes
  • Rating system for community recipes
  • Commenting and feedback options

Project Scope

Our project aims to develop a comprehensive cooking assistant web application that simplifies meal planning, recipe management, and grocery shopping for home users. The application will focus on intuitive design and practical features that address real-world cooking challenges.
